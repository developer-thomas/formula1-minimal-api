# Go into the repository

# Install dependencies
$ npm install

## Endpoints

### 2021 F1 Edition

```
`/teams`: Show all the F1 Teams
`/pilots`: Show all the F1 Pilots
`/pilots/:id`: Filter an specific pilot by its ID
`/teams/:id`: Filter an specific team by its ID
```
